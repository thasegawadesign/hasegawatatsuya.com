## コメント

有用で美しいものが「自在」に創れる状態を目指して、Webについて勉強しています。クリエイティブワーク、モノづくりの達人が目標です。

ホームページ制作の実務においては、想定される多様な訪問者のニーズに応えることを第一に考えます。情報設計、デザイン、コーディングの各工程でアクセシビリティ、ユーザビリティ、審美性、セキュリティ、パフォーマンス、保守性、堅牢性などWebに求められる基本的な性質を認知、構造、表現、技術などの観点から横断的かつ多面的に配慮しながら設計・実装しています。商売道具としてのホームページが果たす役割に持続的に応えられるよう、安定した品質の成果物を届けることが私の役割だと捉えています。さらに、ホームページ公開、運用開始後も継続的に改善を重ね、ホームページを「育てる」姿勢で仕事に取り組みます。加えて、将来的には空間UIや没入型体験など、新しいWeb表現への応用も視野に入れています。
